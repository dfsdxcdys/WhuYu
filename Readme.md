## “武汉大学二手交易平台” 使用指南
1、运行main.py，连接服务器：在终端（cmd）中输入python main.py指令
2、运行login.html
3、输入默认账号admin，密码123456，登录
4、进入主页，下滑至功能板
5、点击“物品发布”按钮，进入物品发布页面，填写信息并发布
6、返回首页地图查看已发布的物品
7、点击想要的物品可将其添加进购物车
8、其他操作
9、完成后退出登录
